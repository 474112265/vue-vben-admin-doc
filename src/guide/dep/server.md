# 测试服务器

在项目`/test/server`内有简单的 node.js 后台接口服务，用 koa.js 实现

## API 接口服务

用于简单测试真实接口

### 运行

```bash

cd ./test/server/api

yarn

yarn start

```

## 上传接口服务

用于简单测试上传组件

### 运行

```bash

# 上传的的文件也位于该文件夹下
cd ./test/server/upload

yarn

yarn start

```

启动后就可以测试上传组件

## websocket 接口服务

用于简单测试 websocket

### 运行

```bash

cd ./test/server/websocket

yarn

yarn start

```

启动后可以测试示例内的 websocket 服务
